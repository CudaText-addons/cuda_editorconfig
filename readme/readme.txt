plugin for CudaText.
supports EditorConfig system in editor.
it reads/applies EditorConfig files for all opened files.
supports the following options:

- indent_style
- indent_size
- tab_width
- end_of_line
- charset
- trim_trailing_whitespace (beginning with CudaText 1.77.4)
- insert_final_newline (beginning with CudaText 1.77.4)
